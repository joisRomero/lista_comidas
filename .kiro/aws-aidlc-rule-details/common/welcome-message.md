# 👋 Welcome to AI-DLC (AI-Driven Development Life Cycle)!

I'll guide you through an adaptive workflow that tailors itself to your project's needs.

## The Five-Phase Lifecycle

```
  🔵 INCEPTION ─── Planning & Architecture (WHAT and WHY)
        ↓
  🟢 CONSTRUCTION ─ Design, Implementation & Test (HOW)
        ↓
  🟡 OPERATIONS ─── Release & Operational Readiness
        ↓
  🟣 CLOSURE ────── Archival & Handoff
        ↓
  🔄 CHANGE MGMT ─ Re-entry for Closed Projects (if needed)
```

## What Happens Next

1. **Workspace analysis** — new or existing project?
2. **Requirements gathering** — clarifying questions in `.md` files
3. **Execution plan** — which stages run and why (you review and approve)
4. **Build** — code generation with checkpoints at each stage
5. **Delivery** — working code with documentation and tests

**Key**: Each stage is independently evaluated. Simple changes skip unnecessary stages. You approve before we proceed.

Let's begin!
